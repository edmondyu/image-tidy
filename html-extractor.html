Extract Text from HTML

Extract structured text from HTML/XHTML files. All processing is done locally.

1. Select HTML/XHTML Files & Tags to Preserve

You can select one or more files, and you can repeat the selection to add more.

Preserve tags:

Only checked tags will be preserved as HTML in the output. All other tags are removed but their text is kept. For headings, unchecking <h1>–<h6> will downgrade them to plain text.
